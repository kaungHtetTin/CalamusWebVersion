စားသောက်ဆိုင်‌တွင် အော်ဒါမှာယူခြင်း

하림 : 저기요.
ဟာရင်မ် : ဒီမှာ။
웨이터 : 네, 주문하세요.
Waiter : ဟုတ်ကဲ့ မှာပါ။
하림 : 김치찌개 하나 주세요.
ဟာရင်မ် : ကင်မ်ချီချဉ်ဟင်း တစ်ပွဲပေးပါ။
웨이터 : 네. 알겠습니다.
Waiter : ဟုတ်ကဲ့ပါ။
하림 : 저기요. 김치찌개 매워요?
ဟာရင်မ် : ဒီမှာရှင့် ကင်မ်ချီချဉ်ဟင်းက စပ်လား။
웨이터 : 네. 저희 김치찌개는 많이 매워요.
Waiter : ဟုတ်ကဲ့ ကျွန်တော်တို့ ကင်မ်ချီချဉ်ဟင်းက အရမ်း စပ်ပါတယ်။
하림 : 진짜요?
ဟာရင်မ် : တကယ်လား။
웨이터 : 네. 된장찌개는 어때요? 된장찌개도 맛있어요.
Waiter : ဟုတ် ပဲငပိဟင်းရည်ဆိုရင်ရော ဘယ်လိုလဲခင်ဗျ? ပဲငပိဟင်းရည်လည်း အရသာရှိပါတယ်။
하림 : 또 뭐 있어요?
ဟာရင်မ် : ထပ်ပြီး ဘာရှိသေးလဲ။
웨이터 : 여기 메뉴판이요.
Waiter : ဒီမှာ menuကဒ်ပါ။
하림 : 흠… 김치찌개 조금 덜 매운 거 없어요?
ဟာရင်မ် : Hmm....ကင်မ်ချီချဉ်ဟင်းက နည်းနည်းဘဲစပ်တာမရှိဘူးလား။
웨이터 : 주방에 물어볼게요.
Waiter : စားဖိုဆောင်ကို မေးပေးပါ့မယ်။
하림 : 네. 감사합니다.
ဟာရင်မ် : ဟုတ်ကဲ့ ကျေးဇူးတင်ပါတယ်။
