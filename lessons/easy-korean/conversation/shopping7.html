စာအုပ်ဝယ်ယူခြင်း

손님 : 저기요. 이 책을 찾고 있는데 못 찾겠어요.
ဝယ်သူ : ဒီမှာခင်ဗျ။ ဒီစာအုပ်ရှာနေတာ ရှာလို့မတွေ့လို့ပါ။
점원 : 이 책은 지금 없어요. 이 책이 인기가 많아서 다 팔렸어요.
ဝန်ထမ်း : ဒီစာအုပ်က အခုမရှိသေးဘူး။ ဒီစာအုပ်က နာမည်ကြီးလို့ အကုန် ရောင်းထွက်သွားပြီရှင့်။
손님 : 그럼 언제 살 수 있어요?
ဝယ်သူ : အဲ့ဒါဆို ဘယ်တော့လောက် ဝယ်လို့ရမလဲဗျ။
점원 : 아직 모르겠어요. 다른 서점에 있을 수도 있어요.
ဝန်ထမ်း : အခုထိတော့ မသိသေးပါဘူး။ တစ်ခြား စာအုပ်ဆိုင်မှာ ရှိနိုင်ပါတယ်။
손님 : 다른 서점은 못 가요. 여기에서 사야 돼요.
ဝယ်သူ : တစ်ခြား စာအုပ်ဆိုင်တော့ မသွားနိုင်ဘူး။ဒီမှာဘဲ ဝယ်မှရမှာ။
점원: 그럼 기다려야 돼요.
ဝန်ထမ်း : အဲ့ဒါဆိုရင် စောင့်ပေးရပါ့မယ်။
손님 : 언제까지 기다려야 해요?
ဝယ်သူ : ဘယ်အချိန်ထိ စောင့်ရမလဲဗျ။
점원 : 다음 주에 다시 오세요.
ဝန်ထမ်း : နောက်တစ်ပတ်ကျ ထပ်လာခဲ့ပါ။
손님 : 다음 주에는 책을 살 수 있어요?
ဝယ်သူ: နောက်တစ်ပတ်ကျရင် စာအုပ်ကို ဝယ်လို့ရနိုင်လားဗျ။
점원 : 그럴 것 같아요.
ဝန်ထမ်း : အဲ့လိုထင်တာပါဘဲ။
손님 : 확실하지 않아요?
ဝယ်သူ : မသေချာဘူးလားခင်ဗျာ။
점원 : 네. 확실하지 않아요
ဝန်ထမ်း : ဟုတ်ကဲ့ မသေချာပါဘူးရှင့်။
