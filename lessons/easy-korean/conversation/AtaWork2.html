အစည်းအဝေးနဲ့ပတ်သတ်ပြီးပြောဆိုခြင်း

최 대리 : 오늘 회의 준비 다 했어?
တာဝန်ခံချွဲ : ဒီနေ့ အစည်းအဝေးအတွက် အကုန် ပြင်ဆင်ပြီးပြီလား။
선 대리 : 아니, 아직 못 했어.
တာဝန်ခံဆေားန် : အခုထိ မပြီးသေးဘူး။
최 대리: 진짜? 5분 후에 회의 시작해.
တာဝန်ခံချွဲ : တကယ်။ နောက်5မိနစ်နေရင် အစည်းအဝေးစမှာ။
선 대리 : 금방 해.
တာဝန်ခံဆေားန် : အခု လုပ်လိုက်မယ်။
최 대리 : 과장님 지금 회의실 간다.
တာဝန်ခံချွဲ : ဌာနမှူး အခု အစည်းအဝေးခန်း သွားနေတယ်။
선 대리 : 너 먼저 가. 나 금방 갈게.
တာဝန်ခံဆေားန် : နင် အရင်သွားလိုက်။ ငါချက်ချင်း လိုက်ခဲ့မယ်။
최 대리 : 알았어.
တာဝန်ခံချွဲ : ကောင်းပြီ။
최 대리 : 선 대리, 천천히 해.
တာဝန်ခံချွဲ : တာဝန်ခံဆေားန်, ဖြည်းဖြည်းချင်း လုပ်။
선 대리 : 왜? 과장님 화났어?
တာဝန်ခံဆေားန် : ဘာလို့? ဌာနမှူး ဒေါသထွက်နေလို့လား။
최 대리 : 아니. 과장님 아파서 병원 가셨어.
တာဝန်ခံချွဲ : မဟုတ်ဘူး ဌာနမှူးက နေမကောင်းလို့ ဆေးရုံသွားလိုက်ပြီ။
선 대리 : 진짜? 그럼 회의는?
တာဝန်ခံဆေားန် : တကယ်။ အဲ့ဒါဆို အစည်းအဝေးကရော။
최 대리 : 회의는 내일 할 거야.
တာဝန်ခံချွဲ : အစည်းအဝေးက မနက်ဖြန်လုပ်မှာ။
