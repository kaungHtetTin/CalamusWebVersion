သူငယ်ချင်းအိမ် သွားလည်ခြင်း

현수 : 들어와.
ဟီယေားန်ဆူ : ဝင်လာလေ။
혜진 : 우와! 집 예쁘다!
ဟယ်ဂျင်း : wow! အိမ်က လှလိုက်တာ!
현수 : 그래? 고마워. 다른 애들은?
ဟီယေားန်ဆူ : ဟုတ်လား ကျေးဇူးပါ တစ်ခြားလူတွေရော။
혜진 : 금방 올 거야. 슈퍼 갔어.
ဟယ်ဂျင်း : အခုလာတော့မှာ စတိုးဆိုင် သွားနေတာ
현수 : 여기 앉아.
ဟီယေားန်ဆူ : ဒီမှာ ထိုင်ပါ။
혜진 : 이거 뭐야? 네가 다 만들었어?
ဟယ်ဂျင်း : ဒါက ဘာလဲ? နင် အကုန်လုပ်ထားတာလား။
현수 : 응. 나 요리 잘해.
ဟီယေားန်ဆူ : အင်း ငါ ဟင်းကောင်းကောင်းချက်တတ်တယ်။
혜진 : 멋지다. 인테리어도 멋져. 그리고 음식도 맛있겠다.
ဟယ်ဂျင်း : မိုက်လိုက်တာ အိမ်တွင်းအပြင်အဆင်လည်း မိုက်တယ် ပြီးတေားအစာတွေကလည်း စားလို့ကောင်းမဲ့ပုံဘဲ။
현수 : 집들이 오늘이 처음이야.
ဟီယေားန်ဆူ : အိမ်တက်ပွဲက ဒီနေ့ ပထမဆုံးဘဲ။
혜진 : 진짜? 우리가 처음이야? 집 진짜 좋다.
ဟယ်ဂျင်း : တကယ်? ငါတို့က ပထမဆုံးလား? အိမ်က တကယ်ကောင်းတယ်ဟ။
현수 : 다행이다. 배고프지? 먼저 먹을래?
ဟီယေားန်ဆူ : တော်ပါသေးတယ် ဗိုက်ဆာနေပြီမလား? အရင်စားမလား။
혜진 : 아니. 기다릴게. 애들 왜 안 오지?
ဟယ်ဂျင်း : တော်ပါပြီ စောင့်လိုက်မယ် သူငယ်ချင်းတွေက ဘာလို့မလးတာလဲ။
