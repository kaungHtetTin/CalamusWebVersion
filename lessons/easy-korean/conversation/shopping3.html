ဖိနပ်ဝယ်ယူခြင်း

미선 : 와! 이 구두 예쁘다. 그런데 이 구두 다른 색깔은 없어요?
မီဆေားန် : ဝါး! ဒီဖိနပ်လေး လှလိုက်တာ။ဒါနဲ့ ဒီဖိနပ်က တခြား အရောင် မရှိဘူးလား။
점원 : 네. 이 색깔밖에 없어요.
ဆိုင်ဝန်ထမ်း : ဟုတ်ကဲ့ ဒီအရောင်ဘဲ ရှိတာပါ။
미선 : 이거 신어 봐도 돼요?
မီဆေားန် : ဒါလေး စီးကြည့်လို့ရမလား။
점원 : 네. 신어 보세요. 사이즈가 어떻게 되세요?
ဆိုင်ဝန်ထမ်း : ရပါတယ် စီးကြည့်ပါ။ size က ဘယ်လောက်လဲရှင့်။
미선 : 230mm요.
မီဆေားန် : 230mmပါ။
점원 : 아, 230mm는 없어요.
ဆိုင်ဝန်ထမ်း : Ah 230mmက မရှိဘူးရှင့်။
미선 : 그럼 235mm는 있어요?
မီဆေားန် : အဲ့ဒါဆို 235ရော ရှိလား။
점원 : 네. 있어요. 잠시만요.
ဆိုင်ဝန်ထမ်း : ဟုတ် ရှိပါတယ်။ ခဏလေးနော်။
점원 : 여기요. 신어 보세요.
ဆိုင်ဝန်ထမ်း : ဒီမှာပါ။ စီးကြည့်ကြည့်ပါ။
미선 : 조금 커요. 그래도 마음에 들어요.
မီဆေားန် : နည်းနည်း ကြီးနေတယ်။ ဒါပေမယ့်လည်း ကြိုက်တယ်။
점원 : 정말 예뻐요.
ဆိုင်ဝန်ထမ်း : တကယ် လှတာဘဲ။
미선 : 이거 살게요. 얼마예요?
မီဆေားန် : ဒါလေး ဝယ်မယ်နော် ဘယ်လောက်လဲ။
