ပရိဘောဂဝယ်ယူခြင်း

손님 : 이 침대 얼마예요?
ဝယ်သူ : ဒီ ကုတင် ဘလောက်လဲ။
점원 : 이건 오백만 원이에요.
ဆိုင်ဝန်ထမ်း : ဒါက ဝမ်သိန်းငါးဆယ်ပါ။
손님 : 비싸네요. 더 싼 거 있어요?
ဝယ်သူ : ဈေးကြီးတယ်နော်။ ပိုဈေးနည်းတဲ့ဟာ ရှိလား။
점원 : 네. 있어요. 그럼 이건 어떠세요?
ဆိုင်ဝန်ထမ်း : ဟုတ်ကဲ့ ရှိပါတယ်။ ဒါဆို ဒီဟာ ဘယ်လိုလဲခင်ဗျ။
손님 : 이건 그것보다 사이즈가 좀 작네요. 가격이 더 싸요?
ဝယ်သူ : ဒီဟာက အဲ့ဟာထက် sizeနည်းနည်း သေးတယ်နော်။ဈေးကပိုနည်းလား။
점원 : 네. 아주 싸요. 사백구십만 원이에요.
ဆိုင်ဝန်ထမ်း : ဟုတ်ကဲ့ အရမ်း ဈေးနည်းပါတယ်။ ဝမ်49သိန်းပါ။
손님 : 별로 안 싸네요. 세일 안 해요?
ဝယ်သူ : သိပ် မနည်းဘူးဘဲ။ saleမရှိဘူးလား။
점원 : 지금은 세일 안 해요.
ဆိုင်ဝန်ထမ်း : အခုတော့ saleမရှိသေးပါဘူး။
손님 : 언제 세일해요?
ဝယ်သူ : ဘယ်တော့ saleရှိမှာလဲ။
점원 : 아직은 계획이 없어요. 세일하면 연락드릴게요.
ဆိုင်ဝန်ထမ်း : အခုတော့ အစီအစဉ်မရှိသေးပါဘူး။saleလုပ်တဲ့အချိန် ဆက်သွယ်ပေးပါ့မယ်။
손님 : 네. 침대 배달해 주세요?
ဝယ်သူ : ဟုတ်။ ကုတင်က deliveryရှိလား။
점원 : 네. 배달해 드려요.
ဆိုင်ဝန်ထမ်း : ဟုတ်ကဲ့ deliveryရှိပါတယ်။
