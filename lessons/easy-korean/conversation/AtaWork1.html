အချိန်ပိုဆင်းခြင်း

미화 : 여보, 오늘 언제 와요?
မီဟွာ : ယောင်္ကျား, ဒီနေ့ ဘယ်အချိန် ပြန်လာမှာလဲ။
성일: 오늘 늦을 거예요.
ဆေားင်အီးလ် : ဒီနေ့ နောက်ကျမှာ။
미화 : 또 야근해요?
မီဟွာ : ထပ်ပြီး ညဆိုင်းဆင်းရတာလား။
성일 : 네.
ဆေားင်အီးလ် : ဟုတ်တယ်
미화 : 아휴.
မီဟွာ : ဟူ။
성일 : 왜요?
ဆေားင်အီးလ် : ဘာလို့လဲ။
미화 : 오늘은 일찍 와요. 네?
မီဟွာ : ဒီနေ့တော့ စောစောပြန်လာပါ ဟုတ်ပြီလား။
성일 : 왜요?
ဆေားင်အီးလ် : ဘာလို့လဲ။
미화 : 진짜 몰라요?
မီဟွာ : တကယ် မသိဘူးလား။
성일 : 네. 몰라요.
ဆေားင်အီးလ် : ဟုတ်တယ် မသိဘူး။
미화 : 오늘 우리 결혼기념일이에요.
မီဟွာ : ဒီနေ့ ကျွန်မတို့ မင်္ဂလာနှစ်ပတ်လည်နေ့လေ။
성일 : 네? 오늘이 며칠이에요?
ဆေားင်အီးလ် : ဟမ်? ဒီနေ့ ဘယ်နှရက်နေ့လဲ။
미화 : 2월 23일이요.
မီဟွာ : 2လ23ရက်နေ့။
성일 : 우리 결혼기념일은 2월 25 일이에요.
ဆေားင်အီးလ် : ကျွန်တော်တို့ မင်္ဂလာနှစ်ပတ်လည်နေ့က 2လ25ရက်နေ့လေ။
미화 : 네? 아… 미안해요.
မီဟွာ : ဟင်။ Ah... တောင်းပန်ပါတယ်။
성일 : 오늘 야근해야 돼요. 25일 에는 일찍 갈게요.
ဆေားင်အီးလ် : ဒီနေ့တော့ ညဆိုင်းဆင်းမှရမယ်။ 25ရက်နေ့ကျရင် စောစောပြန်လာမယ်။
미화 : 네.
မီဟွာ : ဟုတ်ပြီ။
