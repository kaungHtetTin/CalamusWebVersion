ဖွင့်ဟဝန်ခံခြင်း

준호 : 미선아, 지금 시간 있어?
ဂျွန်ဟို : မီဆေားန် အခု အချိန်ရှိလား။
미선 : 왜?
မီဆေားန် : ဘာလို့လဲ။
준호 : 나 할 말이 있어.
ဂျွန်ဟို : ငါပြောစရာရှိတယ်။
미선 : 뭔데?
မီဆေားန် : ဘာလဲ။
준호 : 나 너 좋아해.
ဂျွန်ဟို : ငါနင့်ကို သဘောကျတယ်။
미선 : 장난치지 마.
မီဆေားန် : နောက်မနေနဲ့။
준호 : 장난 아니야. 진심이야. 친구 말고 남자 친구 하고 싶어.
ဂျွန်ဟို : နောက်နေတာမဟုတ်ဘူး။ အတည်ပြောနေတာ။ သူငယ်ချင်းမဟုတ်ဘဲ ချစ်သူကောင်လေးဖြစ်ချင်တယ်။
미선 : 정말이야?
မီဆေားန် : တကယ်ကြီးလား။
준호 : 응. 너 처음 봤을 때부터 좋아했어.
ဂျွန်ဟို : အင်း နင့်ကို ပထမဆုံး တွေ့ကတည်းက သဘောကျမိသွားတာ။
미선 : 정말 몰랐어.
မီဆေားန် : တကယ် မသိခဲ့ဘူး။
준호 : 너는 나 어때?
ဂျွန်ဟို : နင်ရော ငါ့ကို ဘယ်လိုလဲ။
미선 : 사실 나도 너 좋아하고 있었어.
မီဆေားန် : တကယ်တော့ ငါလည်း နင့်ကို သဘောကျနေတာ။
준호 : 정말? 고마워.
ဂျွန်ဟို : တကယ်။ ကျေးဇူးဘဲ။
