ဆရာ့ကိုမေးမြန်းခြင်း

성일 : 선생님, 질문 있어요.
ဆေားင်အီးလ် : ဆရာ၊မေးစရာရှိတယ်။
선생님 : 질문?질문하세요.
ဆရာ : မေးစရာ။မေးပါ။
성일 : 저희 방학 언제부터예요?
ဆေားင်အီးလ် : ကျွန်တော်တို့ ကျောင်းပိတ်ရက်ရှည်က ဘယ်တော့လောက်စမှာလဲခင်ဗျ။
선생님 : 방학? 방학은 7월부터예요.
ဆရာ : ကျောင်းပိတ်ရက်ရှည်? ကျောင်းပိတ်ရက်ရှည်က 7လပိုင်းမှ စမှာလေ။
성일 : 그럼 언제까지예요?
ဆေားင်အီးလ် : အ့ဒါဆို ဘယ်အထိပိတ်မှာလဲဗျ။
선생님 : 8월까지요.
ဆရာ : 8လပိုင်းအထိ။
성일 : 방학 때 저희 학교 나와요?
ဆေားင်အီးလ် : ကျောင်းပိတ်ရက်ရှည်ကျ ကျွန်တော်တို့ ကျောင်းကို လာကမှာလား။
선생님 : 네. 고3은 학교 나와요.
ဆရာ : ဟုတ်တယ် အထက်တန်းတတိယနှစ်တွေက ကျောင်းကို လာရမှာ။
성일 : 그럼 도시락 싸 와요?
ဆေားင်အီးလ် : အဲ့ဒါဆို ထမင်းဘူး ဝယ်ရမှာလား။
선생님 : 진석진! 다시 들어와요. 땡땡이는 안 돼요.
ဆရာ : ဂျင်ဆော့ဂျင်! ပြန်လာခဲ့ အတန်းလစ်လို့မရဘူး။
성일 : 선생님!
ဆေားင်အီးလ် : ဆရာ!
선생님 : 이제 그만. 수업 시작할게요.
ဆရာ : အခု ဒီလောက်ဘဲ အတန်းစမယ်။
