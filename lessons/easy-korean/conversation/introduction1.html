ကိုယ့်ကိုကိုယ် မိတ်ဆက်ခြင်း

제니 : 안녕하세요.
ဂျဲနီ : မင်္ဂလာပါ။
혜진 : 네, 안녕하세요.처음 뵙겠습니다.
ဟဲဂျင် : ဟုတ် မင်္ဂလာပါ။ ပထမဆုံးတွေ့ဆုံခြင်းပါ။
제니 : 반갑습니다. 제 이름은 유제니예요.
ဂျဲနီ : တွေ့ရတာ ဝမ်းသာပါတယ်။ ကျွန်မ နာမည် က ယူဂျဲနီပါ။
혜진 : 네,반갑습니다. 제 이름은 김혜진이에요.
ဟဲဂျင် : ဟုတ် တွေ့ရတာဝမ်းသာပါတယ်။ ကျွန်မ နာမည်က ကင်မ်ဟဲဂျင်ပါ။
제니 : 어디에 사세요?
ဂျဲနီ : ဘယ်မှာ နေတာလဲရှင့်။
혜진 : 저는 서울에 살아요. 제니 씨는요?
ဟဲဂျင် : ကျွန်မကတော့ ဆိုးလ်မှာ နေတာပါ။
제니 : 저도 서울에 살아요. 그런데 고향은 부산이에요.
ဂျဲနီ :ကျွန်မရော ဆိုးလ်မှာဘဲ နေတယ်။ဒါပေမယ့် မွေးရပ်မြေကတော့ ဘူဆန်ပါ။
혜진 : 학생이세요?
ဟဲဂျင် :ကျောင်းသူလားရှင့်။
제니 : 네, 저는 한국대학교에 다녀요. 혜진 씨도 학생이에요?
ဂျဲနီ : ဟုတ်ကဲ့ ကျွန်မက ဟန်းဂုတက္ကသိုလ် မှာ တက်ရောက်နေတာပါ။ ဟဲဂျင်ကရော ကျောင်းသူပဲလားရှင့်။
혜진 : 아니요. 저는 회사원이에요.
ဟဲဂျင် : မဟုတ်ပါဘူး။ ကျွန်မက ကုမ္ပဏီဝန်ထမ်း ပါ။
제니 : 무슨일을 하세요?
ဂျဲနီ : ဘာအလုပ် လုပ်တာလဲရှင့်။
혜진 : 저는 컴퓨터 엔지니어예요.
ဟဲဂျင် : ကျွန်မက ကွန်ပျူတာ အင်ဂျင်နီယာပါ။
