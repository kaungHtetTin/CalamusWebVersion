손님 : 토너하고 로션 추천해 주세요.
ဝယ်သူ : တိုနာနဲ့ lotion ညွှန်းပေးပါ။
점원 : 네. 피부가 지성이세요? 건성이세요?
ဆိုင်ဝန်ထမ်း: ဟုတ်ကဲ့ပါ။ အသားအရေက အဆီပြန်တဲ့ အသားအရေလား။ခြောက်တဲ့အသားအရေလားခင်ဗျာ။
손님: 건성이에요.
ဝယ်သူ : ခြောက်တဲ့အသားရေမျိုးပါ။
점원 : 그럼 이거 써 보세요. 촉촉한 로션이에요.
ဆိုင်ဝန်ထမ်း: အဲ့ဒါဆို ဒါလေး သုံးကြည့်ပါ။ စိုစို‌လေးဖြစ်စေတဲ့ lotionပါ။
손님 : 고맙습니다. 이거 세 개 주세요.
ဝယ်သူ : ကျေးဇူးတင်ပါတယ်။ ဒီဟာ သုံးဘူးပေးပါ။
점원 : 두 개밖에 없어요. 다 떨어졌어요.
ဆိုင်ဝန်ထမ်း : နှစ်ဘူး အပြင်မရှိတော့ဘူး။ အကုန် ကုန်သွားပြီဗျ။
손님 : 아, 그래요? 그럼 두 개만 주세요.
ဝယ်သူ : Ah,ဟုတ်လား။ အဲ့ဒါဆို နှစ်ခုဘဲ ပေးလိုက်ပါ။
점원 : 네. 삼만 원입니다.
ဆိုင်ဝန်ထမ်း : ဟုတ်ကဲ့ပါ။ ဝမ်သုံးသောင်းပါ။
손님 : 여기요.
ဝယ်သူ : ဒီမှာပါ။
점원 : 삼만 원 받았습니다. 포인트 적립해 드릴까요?
ဆိုင်ဝန်ထမ်း : ဝမ်သုံးသောင်း လက်ခံရရှိပါတယ်။ pointထည့်ပေးရမလားခင်ဗျ။
손님 : 아니요, 괜찮아요.
ဝယ်သူ : ရတယ် မထည့်တော့ဘူး။
점원 : 네, 영수증 여기 있습니다. 감사합니다.
ဆိုင်ဝန်ထမ်း : ဟုတ်ကဲ့ ပြေစာက ဒီမှာပါ။ ကျေးဇူးတင်ပါတယ်ခင်ဗျ။
손님 : 네, 수고하세요.
ဝယ်သူ : ဟုတ်ကဲ့ တာဝန်ကျေလိုက်ပါဦးနော်။
