Movie Date

정석 : 민아 씨, 어떤 영화 좋아해요?
ဂျောင်ဆော့ : မင်န်အာက ဘယ်လိုရုပ်ရှင်မျိုးကို သဘောကျလဲ။
민아 : 저는 액션 영화 좋아해요.
မင်န်အာ : ကျွန်မက Action movie ကိုကြိုက်တယ်
정석 : 저도 액션 영화 정말 좋아해요.
ဂျောင်ဆော့ : ကျွန်တော်လဲ Action Movie ကိုတကယ်ကြိုက်တယ်
민아 : 재미있는 액션 영화 하나 개봉했어요. 우리 그거 볼래요?
မင်န်အာ : စိတ်၀င်စားဖို့ကောင်းတယ့် Action ကားတစ်ကားရုံတင်နေတယ်။ကျွန်မတို့ အဲ့ဟာ ကြည့်မလား
정석 : 네. 좋아요. 그럼 제가 지금 예매할게요.
ဂျောင်ဆော့ : အင်းကောင်းပြီ အဲ့တာဆို ကျွန်တော် အခု ကြိုတင်မှာလိုက်မယ်။
민아 : 네. 고마워요.
မင်န်အာ : ဟုတ်ကဲ့ကျေးဇူးပါ။
정석 : 그럼 영화 보기 전에 밥 먹을까요?
ဂျောင်ဆော့ : အဲ့တာဆို ရုပ်ရှင်မကြည့်ခင်ထမင်းစားကြမလား။
민아 : 네. 뭐 먹을까요? 정석 씨 좋아하는 삼겹살 먹을까요?
မင်န်အာ : ဟုတ်ပြီ ဘာစားကြမလဲ? ဂျောင်ဆော့ ကြိုက်တဲ့ သုံးထပ်သားစားမလား။
정석 : 삼겹살 먹고 싶은데 냄새날 것 같아요.
ဂျောင်ဆော့ : သုံးထပ်သား စားချင်ပေမဲ့ အနံ့နံမယ် ထင်တယ်။
민아 : 아, 그럼 뭐 먹을까요?
မင်န်အာ : အာ၊ဒါဆိုရင် ဘာစားမလဲ။
정석 : 햄버거 어때요?
ဂျောင်ဆော့ : ဟမ်ဘာဂါဆိုရင် ဘယ်လိုလဲ။
민아 : 좋아요. 햄버거 먹으러 가요.
မင်န်အာ : ကောင်းပြီ။ဟမ်ဘာဂါ စားဖို့ သွားကြရအောင်။
