အဝတ်အစားဝယ်ယူခြင်း

점원 : 어서 오세요.
ဆိုင်ဝန်ထမ်း : ကြွပါရှင့်။
상미 : 네. 안녕하세요.
ဆန်းမီ : ဟုတ် မင်္ဂလာပါ။
점원 : 뭐 찾으세요?
ဆိုင်ဝန်ထမ်း : ဘာရှာနေတာလဲရှင့်။
상미 : 치마요.
ဆန်းမီ : စကပ်ပါ။
점원 : 치마는 여기에 있어요. 여기 보세요.
ဆိုင်ဝန်ထမ်း : စကပ်က ဒီဘက်မှာရှိပါတယ်။ ဒီမှာ ကြည့်ကြည့်ပါ။
상미 : 네. 이거 다른 색깔은 없나요?
ဆန်းမီ : ဟုတ် ဒါလေးက တစ်ခြားအရောင် မရှိဘူးလားရှင့်။
점원 : 이건 검은색이랑 빨간색밖에 없어요.
ဆိုင်ဝန်ထမ်း : ဒါက အနက်ရောင်နဲ့ အနီရောင်အပြင် မရှိပါဘူး။
상미 : 이거 입어 볼 수 있어요?
ဆန်းမီ : ဒါလေး ဝတ်ကြည့်လို့ရလား။
점원 : 네. 입어 보세요.
ဆိုင်ဝန်ထမ်း : ရပါတယ် ဝတ်ကြည့်ပါရှင်။
상미 : 사이즈가 작아요. 더 큰 거 있어요?
ဆန်းမီ : sizeက သေးနေတယ်။ပိုကြီးတဲ့ဟာ မရှိဘူးလား
점원 : 네, 어떤 사이즈요?
ဆိုင်ဝန်ထမ်း : ဟုတ် ဘယ်sizeလဲရှင့်။
상미 : 허리 27사이즈요.
ဆန်းမီ : ခါး 27 sizeပါ။
