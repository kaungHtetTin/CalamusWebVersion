စာမေးပွဲဖြေနိုင်လားမေးမြန်းခြင်း

현 : 야, 시험 잘 봤어?
ဟီယေားန်အာ : ဟဲ့ စာမေးပွဲ ကောင်းကောင်းဖြေခဲ့လား။
현석 : 아니. 완전히 망쳤어.
ဟီယေားန်ဆော့ : No.လုံးဝ မဖြေနိုင်ခဲ့ဘူး။
현아 : 진짜? 다행이다.
ဟီယေားန်အာ : တကယ်။ တော်ပါသေးတယ်။
현석 : 왜?
ဟီယေားန်ဆော့ : ဘာလို့လဲ။
현아 : 나도 망쳤어.
ဟီယေားန်အာ : ငါလည်းမဖြေနိုင်ခဲ့ဘူး။
현석 : 진짜? 아휴. 나 이거 하나 틀렸어. 너 이거 맞았어?
ဟီယေားန်ဆော့ : တကယ်။ ဟူ ငါဒီဟာ တစ်ခု မှားသွားတယ်။ နင် ဒီဟာ မှန်လား။
현아 : 뭐? 하나?
ဟီယေားန်အာ : ဘာ။ တစ်ခု။
현석 : 응. 이거 하나 틀렸어. 그래서 백 점이 아니야.
ဟီယေားန်ဆော့ : အင်း ဒီဟာ တစ်ခု မှားသွားတာ အဲ့တာကြောင့် အမှတ်100မဖြစ်တော့ဘူး။
현아 : 너 맞고 싶어?
ဟီယေားန်အာ : နင်ရိုက်ခံချင်နေတာလား။
현석  아니. 나 다음 달에는 백 점 맞을 거야. 나 간다.
ဟီယေားန်ဆော့ : no. ငါ နောက်လကျရင် အမှတ်100ရမှာ။ ငါသွားပြီ။
