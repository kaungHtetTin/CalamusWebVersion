အလုပ်ရခြင်း

선미 : 선배! 취업 축하해요.
ဆေားန်မီ : စီနီယာ! အလုပ်ဝင်ခွင့်ရတာ ဂုဏ်ပြုပါတယ်။
선배 : 고마워. 너는?
စီနီယာ : ကျေးဇူးပါဘဲ နင်ရော။
선미: 취업 너무 힘들어요.
ဆေားန်မီ : အလုပ်ရတာဖို့ အရမ်းပင်ပန်းတယ်။
선배 : 이력서 냈어?
စီနီယာ : CVဖောင်တင်ထားလား။
선미 : 네. 진짜 많이 냈어요.
ဆေားန်မီ : ဟုတ် အများကြီး တင်ထားတယ်။
선배 : 한 군데도 연락 안 왔어?
စီနီယာ : တစ်နေရာကမှ ဆက်သွယ်မလာဘူးလား။
선미: 네. 선배, 저 술 사 주세요.
ဆေားန်မီ : ဟုတ် စီနီယာ, ကျွန်မကို အရက်ဝယ်ပေးပါ။
선배 : 그래. 술 마시자.
စီနီယာ : ကောင်းပြီ အရက်သောက်ရအောင်။
선미: 고기도 사 주세요.
ဆေားန်မီ : အသားလည်း ဝယ်ကျွေးပါ။
선배 : 고기? 나 아직 백수야. 돈 없어.
စီနီယာ : အသား။ ငါ အခုထိ အလုပ်လက်မဲ့ဟ ပိုက်ဆံမရှိဘူး။
선미 : 저는 한 군데도 연락 안 왔어요. 슬퍼요.
ဆေားန်မီ : ကျွန်မကို တစ်နေရာကမှ ဆက်သွယ်မလာဘူး ဝမ်းနည်းတယ်။
선배 : 알았어. 고기 먹자. 그리고 술도 마시자.
စီနီယာ : အင်းပါ အသားစားရအောင် ပြီးတော့ အရက်လည်း သောက်ရအောင်။
선미 : 네! 좋아요!
ဆေားန်မီ : ဟုတ်ကဲ့
