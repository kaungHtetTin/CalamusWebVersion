ဟင်းသီးဟင်းရွက်ဝယ်ယူခြင်း

수지 : 안녕하세요.
ဆူဂျီ : မင်္ဂလာပါ။
상인 : 어서 오세요.
ဈေးသည် : ကြွပါခင်ဗျာ။
수지 : 아저씨, 양파 어디 있어요?
ဆူဂျီ : ဦးလေးကြီး ၊ကြက်သွန်နီ ဘယ်မှာလဲရှင့်။
상인 : 양파 여기 있어요.
ဈေးသည် : ကြက်သွန်နီ ဒီမှာ ရှိတယ်။
수지 : 얼마예요?
ဆူဂျီ : ဘယ်လောက်လဲ။
상인 : 네 개에 삼천 원이에요.
ဈေးသည် : လေးလုံးကို 3ထောင်ပါ။
수지 : 여덟 개 주세요.
ဆူဂျီ : 8လုံးပေးပါ။
상인 : 네. 더 필요한 건 없어요?
ဈေးသည် : ဟုတ်ပြီ။
수지 : 당근은 어디 있어요?
ဆူဂျီ : မုန်လာဥနီက ဘယ်မှာလဲရှင့်။
상인 : 당근은 없어요. 다 팔렸어요.
ဈေးသည် : မုန်လာဥနီ မရှိဘူး။အကုန် ရောင်းကုန်သွားပါပြီ။
수지 : 아, 그럼 양파만 주세요.
ဆူဂျီ : Ah ,အဲ့ဒါဆို ကြက်သွန်နီဘဲ ပေးပါ။
상인 : 네. 육천 원이에요.
ဈေးသည် : ဟုတ်ကဲ့ ၊၆ထောင်ကျပါတယ်။
