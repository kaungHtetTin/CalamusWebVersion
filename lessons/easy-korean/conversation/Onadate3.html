Park Date

민수 : 오늘 날씨 정말 좋네요.
မင်ဆူ : ဒီနေ့ ရာသီဥတု တကယ် ကောင်းတာဘဲ။
지연 : 네. 그러네요.
ဂျီယေားန် : အင်း ဟုတ်တယ်နော်။
민수 : 날씨도 좋은데, 우리 공원에서 자전거 탈까요?
မင်ဆူ : ရာသီဥတုလည်း ကောင်းနေတော့ ကျွန်တော်တို့ ပန်းခြံထဲမှာ စက်ဘီးစီးကြမလား။
지연 : 저 자전거 없어요.
ဂျီယေားန် : ကျွန်မမှာ စက်ဘီးမရှိဘူး။
민수 : 한강에서 자전거 빌릴 수 있어요.
မင်ဆူ : ဟန်မြစ်မှာ စက်ဘီးငှားလို့ရတယ်။
지연 : 아, 그래요? 그런데 저 자전거 못 타요.
ဂျီယေားန် : Ah ဟုတ်လား။ ဒါပေမယ့် ကျွန်မ စက်ဘီးမစီးနိုင်ဘူး။
민수 : 괜찮아요. 제 뒤에 타세요.
မင်ဆူ : ရပါတယ်။ ကျွန်တော့်အနောက်မှာ ထိုင်ပြီးစီးပါ။
지연 : 정말요? 좋아요. 그럼 한강에 가요.
ဂျီယေားန် : တကယ်လား?ကောင်းပြီ။ဒါဆို ဟန်မြစ်ကို သွားရအောင်။
민수 : 자전거 빌리려면, 한 시간쯤 기다려야 돼요.
မင်ဆူ : စက်ဘီးငှားမယ်ဆိုရင် တစ်‌နာရီလောက် စောင့်ရတယ်။
지연 : 그럼 우리 자전거 타기 전에 점심 먹어요.
ဂျီယေားန် : အဲ့ဒါဆို ကျွန်မတို့ စက်ဘီးမစီးခင် နေ့လည်စာစားရအောင်။
민수 : 네. 라면 먹을까요?
မင်ဆူ : ဟုတ်ပြီ ခေါက်ဆွဲစားမလား။
지연 : 좋아요.
ဂျီယေားန် : အင်း ကောင်းပြီ။
