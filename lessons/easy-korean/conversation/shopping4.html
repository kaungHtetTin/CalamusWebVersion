ကင်မရာဝယ်ယူခြင်း

점원 : 어서 오세요.
ဆိုင်ဝန်ထမ်း : ကြွပါခင်ဗျာ။
소율 : 네. 안녕하세요.
ဆိုယူးရ် : ဟုတ်ကဲ့ပါ မင်္ဂလာပါရှင့်။
점원 : 어떤 거 찾으세요?
ဆိုင်ဝန်ထမ်း : ဘယ်လိုဟာ ရှာနေတာပါလဲခင်ဗျာ။
소율 : 카메라 찾고 있어요.
ဆိုယူးရ် : ကင်မရာရှာနေတာပါ။
점원 : 어떤 카메라 사고 싶으세요?
ဆိုင်ဝန်ထမ်း : ဘယ်လို ကင်မရာကို ဝယ်ချင်တာပါလဲခင်ဗျ။
소율 : 작고 귀여운 카메라 사고 싶어요.
ဆိုယူးရ် : သေးပြီး ချစ်စရာကောင်းတဲ့ ကင်မရာမျိုး ဝယ်ချင်တာပါ။
점원 : 이건 어때요?
ဆိုင်ဝန်ထမ်း : ဒါလေးဆိုရင် ဘယ်လိုခင်ဗျ။
소율 : 좋아요. 가벼워요.
ဆိုယူးရ် : ကောင်းပါတယ်။ ပေါ့တယ်။
점원 : 이것도 작고 가벼워요. 가격도 싸요.
ဆိုင်ဝန်ထမ်း : ဒီဟာ‌လည်း သေးပြီး ပေါ့တယ်။ဈေးလေး နည်းတယ်။
소율 : 이것보다 아까 그게 더 좋아요.
ဆိုယူးရ် : ဒီဟာထက် ခုနက အဲ့ဟာကို ပိုကြိုက်တယ်။
점원 : 그럼 그걸로 드릴까요?
ဆိုင်ဝန်ထမ်း : ဒါဆို အဲ့ဒါလေး ပေးရမလားခင်ဗျ။
소율 : 조금 깎아 주세요.
ဆိုယူးရ် : နည်းနည်းလောက် လျှော့ပေးပါ။
점원 : 삼만 원 깎아 드릴게요.
ဆိုင်ဝန်ထမ်း : ဝမ်သုံးသောင်း လျှော့ပေးပါ့မယ်။
소율 : 더 깎아 주시면 살게요.
ဆိုယူးရ် : ထပ်ပြီး လျှော့ပေးမယ်ဆိုရင် ဝယ်ပါ့မယ်။
