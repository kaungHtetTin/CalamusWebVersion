ညစာချက်စားခြင်း

상미 :오늘 저녁은 참치 김치찌개야.
ဆားင်မီ : ဒီနေ့ ညစာက တူနာကင်မ်ချီချဉ်ဟင်း။
승우 : 우와! 맛있겠다!
ဆင်ဝု : wow! စားလို့ကောင်းမှာဘဲ!
상미 : 오늘은 네가 도와줘.
ဆားင်မီ : ဒီနေ့တော့ နင် ကူပေးလေ။
승우 : 내가? 나 요리 못해.
ဆင်ဝု : ငါ? ငါ ဟင်းမချက်တတ်ဘူး။
상미 : 참치 김치찌개는 진짜 쉬워. 네가 할래?
ဆားင်မီ : တူနာကင်မ်ချီချဉ်ဟင်းက တကယ်လွယ်တယ်။ နင်လုပ်မလား။
승우 : 내가 해도 맛있을까? 그래. 도와줄게.
ဆင်ဝု : ငါလုပ်ရင် အရသာရှိပါ့မလား? အင်း ကူပေးမယ်။
상미 : 먼저 참치 캔 따 줘.
ဆားင်မီ : အရင်ဆုံး တူနာဘူးခွံ ခွာပေး။
승우 : 응. 여기.
ဆင်ဝု : အင်း ဒီမှာ။
상미 : 참치 기름에 김치를 볶아.
ဆားင်မီ : တူနာဆီထဲမှာ ကင်မ်ချီကို ကြော်လိုက်။
승우 : 나 김치 볶음도 좋아해. 그냥 이거 먹자.
ဆင်ဝု : ငါ ကင်ချီကြော်လည်း ကြိုက်တယ် ဒီတိုင်း ဒါဘဲ စားရအောင်။
상미 : 기다려. 금방 해. 이제 물 부어.
ဆားင်မီ : စောင့်နေပါ အခု ပြီးတော့မှာ အခု ရေ‌လောင်းထည့်လိုက်။
승우 : 응. 그리고 뭐 해?
ဆင်ဝု : အင်း ပြီးတော့ ဘာလုပ်ရမှာလဲ။
상미 : 참치 넣어. 끝.
ဆားင်မီ : တူနာထည့်လိုက် ပြီးပြီ။
