ငြင်းဆိုခြင်း

준서 : 주희 씨, 남자 친구 있어요?
ဂျွန်စော : ဂျူဟီ ချစ်သူကောင်လေးရှိလား။
주희 : 아니요. 없어요.
ဂျူဟီ : မရှိပါဘူး။
준서 : 그래요? 그럼 저는 어때요?
ဂျွန်စော : ဟုတ်လား။ အဲ့ဒါဆို ကျွန်တော်ဆိုရင် ဘယ်လိုလဲ။
주희 : 네? 무슨 말이에요?
ဂျူဟီ : ရှင်။ ဘာပြောလိုက်တာလဲ။
준서 : 저 주희 씨 좋아해요. 주희 씨랑 만나고 싶어요.
ဂျွန်စော : ကျွန်တော် ဂျူဟီကို သဘောကျတယ်။ ဂျူဟီနဲ့ တွဲချင်တယ်။
주희 : 네? 우리 아직 한 번밖에 안 만났어요.
ဂျူဟီ : ရှင်။ ကျွန်မတို့ အခုမှ တစ်ကြိမ်ဘဲ တွေ့ရသေးတယ်။
준서 : 알아요. 그래도 전 주희 씨가 너무 좋아요. 주희 씨는 제가 싫어요?
ဂျွန်စော : သိပါတယ်။ ဒါပေမယ့်လည်း ကျွန်တော် ဂျူဟီကို အရမ်း ကြိုက်တယ်။ဂျူဟီက ကျွန်တော့်ကို မုန်းလို့လား။
주희 : 아니요. 싫지 않아요.
ဂျူဟီ : မဟုတ်ပါဘူး။ မမုန်းပါဘူး။
준서 : 그럼 우리 사귀어요.
ဂျွန်စော : အဲ့ဒါဆို ကျွန်တော်တို့ တွဲပြီနော်။
주희 : 저는 아직 준서 씨 잘 몰라요.
ဂျူဟီ : ကျွန်မက အခုထိ ဂျွန်စောကို ကောင်းကောင်း မသိသေးဘူး။
준서 : 천천히 알게 될 거예요.
ဂျွန်စော : ဖြည်းဖြည်းချင်း သိသွားမှာပါ။
주희 : 아니요. 저는 아직 사귀기 싫어요.
ဂျူဟီ : မဟုတ်ဘူး။ ကျွန်မ အခု မတွဲချင်သေးဘူး။
