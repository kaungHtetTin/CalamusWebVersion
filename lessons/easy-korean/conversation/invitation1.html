ညစာစားခြင်း

현우 : 세경아, 내일 바빠?
ဟီယေားန်ဝု : ဆယ်ဂယောင်း မနက်ဖြန် အားလား။
세경 : 내일 선영이 만나.
ဆယ်ဂယောင်း : မနက်ဖြန် ဆောန်ယောင်နဲ့ တွေ့မှာ။
현우 : 그래?
ဟီယေားန်ဝု : ဟုတ်လား။
세경 : 응. 왜? 무슨 일 있어?
ဆယ်ဂယောင်း : အင်း ဘာလို့လဲ? ဘာကိစ္စရှိလို့လဲ?
현우 : 아니, 그냥. 술 한잔하고 싶어서.
ဟီယေားန်ဝု : မဟုတ်ဘူး ဒီတိုင်းဘဲ။ အရက်တစ်ခွက်လောက်သောက်ချင်လို့။
세경 : 그래? 내일 우리도 술 한잔할 거야. 너도 와!
ဆယ်ဂယောင်း : ဟုတ်လား? မနက်ဖြန် ငါတို့ လည်း အရက်သွားသောက်မှာ နင်လည်း လာခဲ့လေ။
현우 : 그래도 돼?
ဟီယေားန်ဝု :အဲ့လိုလုပ်လည်း ရလား?
세경 : 응 와도 돼. 선영이도 괜찮을 거야.
ဆယ်ဂယောင်း : အင်း လာလည်း ရတယ်။ ဆောန်ယောင်းလည်း အဆင်ပြေလောက်မှာပါ။
현우 : 좋아. 그럼 어디로 가면 돼?
ဟီယေားန်ဝု : ကောင်းပြီ။ အဲ့ဒါဆို ဘယ်ကို လာခဲ့ရမလဲ။
세경 : 내일 퇴근 후에 나랑 같이 가자.
ဆယ်ဂယောင်း : မနက်ဖြန် အလုပ်ဆင်းရင် ငါနဲ့ အတူသွားသောက်ရအောင်။
현우 :선영이는?
ဟီယေားန်ဝု :ဆောန်ယောင်းကရော?
세경 : 선영이는 일곱 시까지 술집으로 올 거야.
ဆယ်ဂယောင်း : ဆောန်ယောင်းက 7နာရီအထိအရက်ဆိုင်ကို လာမှာ။
